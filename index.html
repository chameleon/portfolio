About this site

This site serves at my portfolio and and features recent work created in a typical UX Design process.

  • This site is a hand-coded HTML5 responsive site. I use Sublime editor. I use Grunt to process lessc/css and minify css & js.
  • The site showcases modern Flat Design techniques (no gradients, bevels or heavy 3D effects) as well as Material Design aesthetics pioneered at Google (such as slight shadows indicating Z order, slight hover effects, roboto font and source-point animated slide-outs
  • The parallax effect at the top of the page is limited to the title area and is purely for visual fun. I combined it with tilt-shift photo effects and basic 3D action. I do have an idea for turning this into a creative and very thumb-friendly nav element.
  • For dynamic resizing, I'm using html{font-size: 62.5%;} to make each REM = 10px and set all heights to REMs and widths to % of parents.
  • I used minimal media queries in the CSS with breakpoints where the content suggests they shoud be, instead of where device screen widths are trending.
  • Less.js
  • CSS animations used where possible to avoid jQuery hiccups and load on mobile procesors.
  • Font icons vs images.
